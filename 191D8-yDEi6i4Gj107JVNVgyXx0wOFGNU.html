Payment and Cashing

Summary

Ulys is an API to manage cart workflow and offer a specific user experience depending on each payment mode : mobility (RapidScan/MCO), self checkout (SCO), cashier checkout (CCO), etc.

Ulys Cart API houses most of the source codes for all Ulys modes. The tools used are:

  • Java and Groovy are the programming languages for this project
  • Spring Boot as the main framework
  • MongoDB Atlas or [Couchbase](https://www.couchbase.com) to store the data
  • Apache Kafka to publish data that can be consumed
  • Hazelcast to manage distributed cache
  • Karate for test automation
